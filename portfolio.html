1. Epidemiological Compartmental Model with Bayesian Inference

PyMC SciPy NUTS/HMC Bayesian Inference

Reproduced and extended a non-linear epidemiological model with hierarchical Bayesian inference. Implemented ODE-based SEIR dynamics with advanced sampling techniques (NUTS, HMC, custom MCMC) for parameter estimation and uncertainty quantification. Compared multiple model specifications using WAIC and LOO cross-validation.

Key Results: Successfully translated R-based models to PyMC, achieving convergence on complex hierarchical structures and providing robust uncertainty estimates for disease dynamics parameters.

2. Healthcare Professional Publication Matching System

Python SQL Snowflake PMI Analysis

Developed probabilistic matching algorithm to link 3.9M healthcare professionals with 25M+ PubMed publications at Alpha Sophia. Implemented custom scoring functions combining pointwise mutual information (PMI) analysis, k-means clustering, and email auto-complete algorithms.

Key Results: Achieved 103% improvement in match rates, deployed to production in July 2025. Engineered complete SQL/XML database structure for clinical trials integration.
